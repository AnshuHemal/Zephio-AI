# Keyboard Shortcuts

Zephio includes comprehensive keyboard shortcuts to help power users work faster and more efficiently.

## Global Shortcuts

These shortcuts work throughout the application:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` / `Ctrl+K` | Open Command Palette | Quick access to all commands, projects, and settings |
| `⌘N` / `Ctrl+N` | New Project | Create a new project |
| `Escape` | Close / Deselect | Close modals, command palette, or deselect active page |

## Command Palette (`⌘K`)

The command palette provides quick access to:

- **Navigation**: Dashboard, New Project
- **Recent Projects**: Jump to any of your last 5 projects
- **Appearance**: Switch between Light, Dark, and System themes
- **Actions**: Upgrade to Pro, Sign Out

## Project / Canvas Shortcuts

When working on a project:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘Z` / `Ctrl+Z` | Undo | Undo the last page change |
| `⌘⇧Z` / `Ctrl+Shift+Z` | Redo | Redo the last undone change |
| `⌘Y` / `Ctrl+Y` | Redo | Alternative redo shortcut |
| `Escape` | Deselect Page | Deselect the currently selected page |

## Chat Input Shortcuts

When the chat input is focused:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘Enter` / `Ctrl+Enter` | Submit Prompt | Send your message to generate pages |
| `Escape` | Deselect Page | Clear the selected page badge |

## Dashboard Shortcuts

On the dashboard:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘K` / `Ctrl+K` | Open Command Palette | Search projects, navigate, change theme |
| `⌘N` / `Ctrl+N` | New Project | Start a new project |

## Tips

- **Mac vs Windows/Linux**: Shortcuts automatically adapt to your platform. `⌘` (Command) on Mac becomes `Ctrl` on Windows/Linux.
- **Escape Priority**: Pressing `Escape` will close the topmost modal or dialog first, then deselect pages.
- **Command Palette**: The command palette (`⌘K`) is your fastest way to navigate between projects and access features.
- **Undo/Redo**: Changes to pages are tracked in history. Use undo/redo to experiment freely.

## Accessibility

All keyboard shortcuts are designed to work alongside screen readers and other assistive technologies. Interactive elements remain keyboard-navigable using `Tab` and `Enter`.
